# VSCode Extension Developer Resources

## Overview
This folder contains resources used by the EDA Family VSCode extension.

## Contents

### scripts/
- `eda_analysis.py`: Python script template for EDA analyses

### icons/
- `icon.svg`: Main extension icon
- `father-icon.svg`: Father role icon
- `mother-icon.svg`: Mother role icon
- `son-icon.svg`: Son role icon
- `daughter-icon.svg`: Daughter role icon
